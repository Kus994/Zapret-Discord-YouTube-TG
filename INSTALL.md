# KUS Pro — Инструкция по установке

## Быстрая установка (3 шага)

### Шаг 1: Скачать

Скачайте последний релиз: [GitHub Releases](https://github.com/Kus994/Zapret-Discord-YouTube-TG/releases/latest)

### Шаг 2: Распаковать

1. Найдите скачанный файл `.zip`
2. Нажмите правой кнопкой → **Извлечь все**
3. Выберите путь без кириллицы и спецсимволов, например: `C:\KUS Pro`

### Шаг 3: Запустить

1. Найдите файл `run.bat` в папке
2. Нажмите правой кнопкой → **Запуск от имени администратора**
3. Готово!

---

## Требования

- **ОС**: Windows 10/11 (64-bit)
- **Python**: 3.8+ (автоматически проверяется при запуске)
- **PyQt5**: устанавливается автоматически при первом запуске
- **Права администратора**: требуются для Zapret и некоторых функций

---

## Если Python не установлен

1. Скачайте Python с [python.org/downloads](https://www.python.org/downloads/)
2. При установке **обязательно поставьте галочку** "Add Python to PATH"
3. Перезапустите компьютер
4. Запустите `run.bat` снова

---

## Автоматическая установка зависимостей

При первом запуске `run.bat` автоматически:
- Проверяет наличие Python
- Устанавливает PyQt5 (`pip install PyQt5`)
- Запускает приложение

## Структура папки после распаковки

```
KusXXX-main/
├── main.py              # Точка входа

├── widgets.py           # Кастомные виджеты

├── config.json          # Конфигурация (создаётся автоматически)
├── zapret/              # Zapret DPI bypass
│   ├── winws.exe        # Основной бинарник
│   ├── bin/             # Вспомогательные файлы
│   ├── lists/           # Списки хостов
│   └── *.bat            # Пресеты запуска

```

## Сборка .exe (опционально)

Если хотите собрать Standalone-приложение:

1. Установите PyInstaller: `pip install pyinstaller`
2. Запустите `build.bat`
3. Готовый `.exe` появится в папке `dist/`

## Решение проблем

### "Python не найден"
- Установите Python и добавьте в PATH
- Или используйте `run.bat` — он автоматически ищет Python

### "PyQt5 не установлен"
- `run.bat` автоматически установит PyQt5
- Или вручную: `pip install PyQt5`

### "Нет прав администратора"
- Нажмите правой кнопкой на `run.bat` → Запуск от имени администратора

### "Zapret не работает"
- Убедитесь что запустили от имени администратора
- Проверьте наличие файлов в папке `zapret/`
- Откройте вкладку "Zapret → Диагностика"
