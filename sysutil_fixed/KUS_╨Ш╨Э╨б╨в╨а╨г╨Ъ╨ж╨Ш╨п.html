ℹ️ Что это такое и зачем?

KUS v2 - это современная утилита для системного администратора Windows.

Она позволяет:

  • ✓ Очищать системный мусор (временные файлы, кэш)
  • ✓ Мониторить ресурсы (CPU, RAM, диски)
  • ✓ Управлять сетью (netstat, DNS, TCP/IP)
  • ✓ Управлять процессами (список, завершение)
  • ✓ Обновлять Windows и драйверы
  • ✓ И многое другое...

✨ Главное улучшение в V2:

Приложение больше НЕ ЗАВИСАЕТ при выполнении операций!
Весь интерфейс остаётся отзывчивым и красивым. 🎉

💻 Установка

Шаг 1: Python

1 Установите Python 3.10+
  1. Перейдите на https://www.python.org/downloads/
  2. Скачайте последнюю версию Python
  3. Запустите установщик
  4. ⚠ ВАЖНО: Поставьте галочку "Add Python to PATH"
  5. Нажмите "Install Now"

Проверка: Откройте PowerShell и напишите:

python --version

Должно вывести: Python 3.10.x или выше

Шаг 2: Скачайте файлы

2

Скачайте архив sysutil.zip и разархивируйте его куда-нибудь.

Примеры путей:

  • C:\Users\YourName\Desktop\sysutil
  • C:\Users\YourName\Documents\sysutil

Шаг 3: Установите зависимости

3
  1. Откройте PowerShell или CMD
  2. Перейдите в папку sysutil:
    cd путь\к\папке\sysutil
  3. Выполните команду:
    pip install -r requirements.txt
  4. Дождитесь завершения (1-3 минуты)

🎯 Запуск приложения

Способ 1: С правами администратора (правильный) ⭐

  1. Откройте папку где лежит main.py
  2. Удерживайте Shift + кликните правую кнопку на пустом месте
  3. Выберите "Открыть PowerShell здесь"
  4. Напишите:
    python main.py
  5. Windows спросит о правах - нажмите "Да"
  6. Приложение откроется за 4-5 секунд

Способ 2: Без администратора (быстро)

  1. Откройте CMD или PowerShell
  2. Перейдите в папку:
    cd путь\к\sysutil
  3. Запустите:
    python main.py

⚠ Внимание: Некоторые функции не будут работать без администратора!

Способ 3: Создайте ярлык (удобно) 🎁

  1. В папке sysutil кликните правую кнопку на main.py
  2. "Отправить" → "Рабочий стол (создать ярлык)"
  3. Теперь на рабочем столе есть ярлык!

✨ Функции и как их использовать

🧹 ОЧИСТКА СИСТЕМЫ

Что делает:

Удаляет временные файлы, кэш и мусор из системы.

Как использовать:

  1. Нажмите кнопку "🧹 Очистка системы" слева
  2. Поставьте галочки рядом с операциями
  3. Нажмите "Начать очистку"
  4. Смотрите волновой прогресс-бар
  5. Смотрите логирование с цветами:
    • ✓ Зелёный = успешно
    • ⚠ Оранжевый = предупреждение
    • ✗ Красный = ошибка
  6. Увидите красивый отчет о результатах

Результаты:

  • Обычно освобождается: 1-5 ГБ
  • Время выполнения: 2-10 минут
  • UI остаётся отзывчивым: ✓

🌐 СЕТЕВАЯ ДИАГНОСТИКА

Что делает:

Показывает все активные сетевые соединения и позволяет сбросить DNS/TCP/IP.

Кнопки:

  • Показать соединения - таблица всех активных соединений
  • Сбросить DNS - очищает кэш доменов
  • Сбросить TCP/IP - перезагружает сетевой стек

Когда использовать:

  • Интернет медленный → Сбросить DNS кэш
  • Интернет не работает → Сбросить TCP/IP
  • Проверить программы в интернете → Показать соединения

⚙️ МЕНЕДЖЕР ПРОЦЕССОВ

Что делает:

Показывает список всех запущенных процессов и позволяет их завершать.

Как использовать:

  1. Нажмите "⚙️ Менеджер процессов"
  2. Увидите таблицу со всеми процессами
  3. Выберите процесс
  4. Нажмите "Завершить процесс"

⚠ ОСТОРОЖНО! Не завершайте системные процессы:

  • svchost.exe
  • kernel_task
  • explorer.exe

🔄 ОБНОВЛЕНИЯ WINDOWS

Что делает:

Проверяет и устанавливает обновления Windows и драйверов.

Как использовать:

  1. Нажмите "🔄 Обновления Windows"
  2. Нажмите "Проверить обновления"
  3. Выберите какие установить
  4. Нажмите "Установить"
  5. Компьютер может перезагрузиться

💡 Совет: Делайте это в конце дня!

📊 МОНИТОРИНГ РЕСУРСОВ

Что показывает:

  • Использование CPU (%)
  • Использование памяти (%)
  • Использование дисков (%)
  • Температура процессора

Цветная кодировка:

  • Зелёная линия = нормально
  • Оранжевая = высоко
  • Красная = критично

Нормальные значения:

  • CPU 30-50% - нормально
  • CPU 80%+ - что-то грузит
  • RAM 50-70% - нормально
  • RAM 90%+ - нужно закрыть программы

🔧 Решение проблем

❌ ModuleNotFoundError: No module named 'customtkinter'

Решение:

pip install customtkinter

⏱️ Приложение запускается долго (5+ сек)

ℹ ЭТО НОРМАЛЬНО! Python требует времени для запуска.

Используйте ярлык на рабочем столе для удобства.

🔒 Access is denied (отказано в доступе)

Решение:

  1. Откройте PowerShell "Запуск от администратора"
  2. Перейдите в папку:
    cd path\to\sysutil
  3. Запустите:
    python main.py
